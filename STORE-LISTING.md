# Social Cleaner

Application Android de nettoyage de stockage pour les fichiers multimédias des réseaux sociaux.

## Fonctionnalités

- Scan intelligent des fichiers WhatsApp, Telegram, Facebook, Instagram, Snapchat, TikTok, Messenger, Signal
- Filtrage par année
- Suppression sélective par catégorie (Images, Vidéos, Documents, Audio, etc.)
- Interface moderne avec thème sombre
- 100% local — aucune connexion réseau requise

## Build

```bash
# Debug APK
gradle assembleDebug

# Release AAB (Google Play)
gradle bundleRelease
```

## Google Play Store

### Description courte (80 caractères)
Libérez l'espace de votre téléphone en nettoyant les fichiers des réseaux sociaux

### Description longue
Social Cleaner est l'outil ultime pour libérer l'espace de stockage de votre téléphone en analysant et supprimant intelligemment les fichiers multimédias accumulés par vos applications de réseaux sociaux.

Fonctionnalités principales :
• Scan complet de 8+ applications (WhatsApp, Telegram, Facebook, Instagram, Snapchat, TikTok, Messenger, Signal)
• Filtrage par année pour cibler les anciens fichiers
• Catégorisation automatique (Images, Vidéos, Documents, Audio, Stickers, GIFs)
• Suppression sélective par catégorie
• Interface moderne et intuitive
• 100% local — aucune donnée envoyée sur Internet

### Permissions justifiées
- MANAGE_EXTERNAL_STORAGE : Nécessaire pour accéder aux dossiers des applications de réseaux sociaux sur Android 11+
- READ/WRITE_EXTERNAL_STORAGE : Pour lire et supprimer les fichiers multimédias
